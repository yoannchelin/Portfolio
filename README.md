# Yoann Chelin — Portfolio (static)

A lightweight, recruiter‑friendly portfolio (HTML/CSS/JS). Includes:
- CV download button (`/assets/Yoann_Chelin_CV.pdf`)
- Featured project (CafeFlow) + space for a second project
- Contact + GitHub link

## Edit content
Open `index.html` and replace:
- CafeFlow repo link + live demo link
- Optional: add LinkedIn later
- Update text in the hero/experience sections

## Deploy (easy)
### Option 1 — GitHub Pages
1. Repo → Settings → Pages
2. Source: `main` + `/ (root)`
3. Save → you’ll get a public URL

### Option 2 — Vercel
1. Import GitHub repo in Vercel
2. Framework preset: **Other**
3. Deploy
